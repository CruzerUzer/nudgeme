# Default-bilder för seed-aktiviteter

Lägg bildfiler här så får **nya konton** automatiskt en bild på matchande
seed-aktivitet (bilden bäddas in som data-URL vid registrering). Befintliga
konton påverkas inte.

## Namngivning

Filnamnet ska vara aktivitetens **slug** + filändelse. Slug = gemener, svenska
tecken översatta (å/ä→a, ö→o), övriga tecken → bindestreck. Stödda ändelser:
`.jpg`, `.jpeg`, `.png`, `.webp`.

Exempel: aktiviteten _"Tända ljus"_ → filen `tanda-ljus.jpg`.
